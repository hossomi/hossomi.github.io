# hossomi.github.io

## Requirements

- Ruby (last tested with 3.4.7)
  
### Development

Recommended development environment (based solely on what I have been using):

- WSL: Ubuntu
- [`rbenv`](https://github.com/rbenv/rbenv-installer#rbenv-installer) (may require installing system dependencies).

## Running

### Local

Execute `bundle exec jekyll serve` then open http://127.0.0.1:4000
